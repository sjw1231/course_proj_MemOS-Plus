# MemOS-Plus: User-Centric Memory Management for Language Models
As large language models (LLMs) evolve toward more capable agents, effective memory management is crucial for long-term reasoning and personalized interaction. While the MemoryOS framework (Kang et al., 2025) provides a foundational hierarchical architecture, it faces limitations in retrieval transparency, storage efficiency, and the handling of diverse information types. In this paper, we propose MemOS-Plus, an enhanced framework designed to address these gaps. We augment the original architecture by introducing explainable retrieval mechanisms, content-aware memory routing, graph-based context associations, and semantic compression. Experiments show that our approach not only improves the system’s token efficiency but also enhances user trust through interpretable reasoning, offering a more robust solution for long-horizon human-LLM interactions. Our code is available at: https://github.com/hexuan21/course_proj_MemOS-Plus
